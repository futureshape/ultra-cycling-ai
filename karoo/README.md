# Karoo Extension

Placeholder for the Karoo Android extension (future implementation).

This will be a minimal Android app / Karoo Extension that:
- Subscribes to navigation state, ride state, location updates, and sensor data
- Uploads the route once at navigation start
- Sends periodic tick updates to the backend
- Provides manual intake UI (eat/drink logging)
- Displays advice alerts with cooldown handling
- Tolerates intermittent connectivity
